# Changelog

## 1.0.0

Primera versión funcional del plugin.

### Incluye
- estructura base del plugin para WordPress
- integración mediante shortcode
- widget frontend de chat
- panel de ajustes básico
- integración con OpenAI
- integración con Google Calendar
- consulta de disponibilidad
- creación de citas
- documentación inicial de instalación y configuración

### Estado
Versión inicial operativa en desarrollo activo. Requiere pruebas, ajustes y validación adicional antes de considerarse una versión cerrada para producción.
